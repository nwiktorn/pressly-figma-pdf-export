# Third-party notices

## Bundled ICC color profile

`assets/CoatedFOGRA39.icc` — **Coated FOGRA39 (ISO 12647-2:2004)**, created by the
European Color Initiative (ECI, https://www.eci.org) from FOGRA characterization
data. ECI provides its ICC profiles free of charge and permits redistribution
provided the profiles are passed on unmodified. The profile is bundled unchanged
and embedded into exported PDFs as the PDF/X OutputIntent.

This profile is a sensible default for European coated-stock printing. Print shops
that require a different output condition can supply their own profile.
